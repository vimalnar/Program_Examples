GENplus Program: Celsius to Fahrenheit Converter

1. Start of Program
   - Begin the temperature converter program.

2. Input
   - Ask the user to input the temperature in Celsius.

3. Validation Check
   - If the input is not a number, inform the user and ask for the temperature again.
   - If the input is a number, proceed to the conversion.

4. Conversion Logic
   - Convert the temperature from Celsius to Fahrenheit using the formula: Fahrenheit = (Celsius * 9/5) + 32.

5. Output
   - Display the converted temperature in Fahrenheit to the user.

6. Error Handling
   - If the conversion process encounters any errors, inform the user that something went wrong and suggest trying again.

7. End of Program
   - End the temperature converter program.

Example Error Handling Scenarios
1. Non-numeric Input
   - If the user enters a non-numeric value, display 'Invalid input. Please enter a numeric value.' and prompt for input again.
2. Unreasonable Temperature Value
   - If the user enters a temperature less than absolute zero (-273.15 Celsius), display 'Temperature below absolute zero is not possible. Please enter a realistic value.' and ask for the input again.
3. System Errors
   - In case of system-related errors (like memory issues), display 'A system error occurred. Please try again later or contact support.'
4. Unexpected Situations
   - For any other unexpected situations, display 'An unexpected error occurred. Please retry or contact support for assistance.'
